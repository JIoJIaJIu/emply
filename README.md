emply
=====
Для отрисовки карты используется topoJSON и d3.js

###./get.sh [--force]
Скрипт для получения topojson'a из shapefile

Dependencies:  
* wget
* p7zip
* [gdal](http://www.gdal.org)
* node && npm

##Server
* `cd server`
* `npm install`
* `node app.js`
